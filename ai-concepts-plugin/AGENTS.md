# AI Concepts Cheat Sheet

This repo provides an AI concepts cheat sheet covering LLMs, tokens, prompts, agents, subagents, skills, MCP, hooks, memory, RAG, embeddings, hallucination, and temperature.

The full cheat sheet lives at `skills/ai-concepts/SKILL.md` (readable markdown — ignore the YAML frontmatter at the top).

When the user asks what an agent, skill, MCP, or RAG is, asks for AI terminology to be explained, or wants a refresher on how AI tooling concepts fit together, read that file and present the relevant section.
